# Code of Conduct

This project adopts version 2.1 of the **Contributor Covenant**, the industry-standard code of conduct for open-source communities. We expect everyone interacting with the project — in issues, PRs, discussions, and any official channel — to follow it.

The full text lives at the canonical source:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you experience or witness behavior that violates the Contributor Covenant, please report it privately to:

**solomon.aboagye@amalitech.com**

Reports go to the maintainer and stay confidential. Reporters won't face retaliation for reporting in good faith. We'll acknowledge receipt within a few business days and follow up with next steps.

## Enforcement

The maintainer ([@dacostaaboagye](https://github.com/dacostaaboagye)) may take any action they consider appropriate — including a temporary or permanent ban from the project's spaces — in response to behavior that violates this code.
